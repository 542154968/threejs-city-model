# Threejs 城市模型渲染 demo 版本

1. **代码比较乱**
2. **注释少**
3. **没有 Threejs 基础的建议补基础再来看实现**

**注意 1**

- 城市的道路也是建模画出来的
- 如果要道路光效可以让建模把道路贴图或者色值调亮点
- 配上辉光自然而然就发光

**注意 2**

- 因为这个是 demo 模型
- 所以模型的中心点并不是正中央
- 导致你想旋转模型的时候
- 它不是绕着中央旋转的
- 可以在模型工具中调整中心点
- threejs 如何调整还没想到
- 利用了 center 方法模型会全部居中 无效
